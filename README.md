# Maracana Salon Website

## Overview
Maracana Salon Website is a responsive and modern web application designed to provide an easy-to-use interface for customers to book salon services online. The website showcases various services, allows users to make appointments, and provides essential information about the salon.

## Features
- User-friendly interface
- Online appointment booking
- Service listings with detailed descriptions
- Customer reviews and ratings
- Contact information and location map

## Technologies Used
- HTML, CSS, JavaScript
- Front-end frameworks: Bootstrap
- Back-end: Node.js, Express
- Database: MongoDB

## Installation
### Prerequisites
- Node.js
- MongoDB

### Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/louatimohamed499-cpu/maracana-salon-website.git
   ```
2. Navigate into the project directory:
   ```bash
   cd maracana-salon-website
   ```
3. Install the required packages:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```

## Usage
Visit `http://localhost:3000` in your web browser to access the Maracana Salon Website.
